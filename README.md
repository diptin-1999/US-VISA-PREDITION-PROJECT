# US-VISA-PREDITION-PROJECT
I recently worked on a Python-based ML project to predict whether a US visa application will be Certified or Denied using real-world data.
# What I implemented:
Data analysis using Pandas & NumPy
Visualization with Matplotlib, Seaborn, Plotly
End-to-end EDA & feature engineering
Applied Chi-Square test for feature importance
Handled imbalanced data & outliers
Built a classification model
# 📊 Key Insights:
🎓 Education plays a major role (Master’s & PhD → higher approval)
💼 Work experience improves approval (~74%) 
💰 Yearly wage contracts have better success rates 
⚠️ Job training has minimal impact
📉 Dataset is imbalanced (~66% approved)
🛠 Tech Stack:
# Python | Pandas | NumPy | Matplotlib | Seaborn | Plotly | Scikit-learn
